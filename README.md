# Tienda de Reparaciones en Python (Consola)

Implementación de **Listas Enlazadas** para un aplicación de consola en python (sin persistencia de datos).

Permite:

-   Gestión de Usuarios
-   Gestión de Clientes
-   Gestión de Equipos
-   Gestión de Ordenes de Trabajo
-   Gestión de Entregas
-   Login y Logout

## Ejecución

```bash
python Principal.py
```
